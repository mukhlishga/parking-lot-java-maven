# Parking Lot

Please see the Problem Statement for context onboarding

## Requirements

- JDK 8 or above
- Maven 2.4 or above
- Junit 4.13.2 or above

## How to run

1. Build
   ```
   ./bin/setup
   ```

2. Run program in interactive CLI
   ```
   ./bin/parking_lot
   ```

3. Run program with file input
   ```
   ./bin/parking_lot inputFile.txt
   ```
